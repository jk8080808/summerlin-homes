# Summerlin Homes Daily

This is a polished starter site for a Summerlin single-family home search website.

## Built-in filters
- Summerlin, Nevada
- Single-family homes
- For sale and under contract
- Under $1.5M
- Under 3,500 sq ft
- Lot size over 5,500 sq ft

## Compliance-ready fields already built in
- Authorized detail page URL
- Photo URL for public-display media
- Listing firm attribution
- Listing agent name
- Phone or email attribution contact
- MLS/source label
- Consumer-use disclaimer area

## What's included
- Responsive website homepage
- Listing cards and filter bar
- Summerlin village filter
- Email alert signup section
- Map placeholder section
- Daily GitHub Actions refresh at 8:00 AM HST
- Sample updater script for a future IDX / MLS / RESO feed

## Beginner-friendly first steps
1. Unzip the folder.
2. Open `index.html` in your browser to preview the site.
3. If you want it online, create a GitHub account.
4. Create a new repository and upload these files.
5. Turn on GitHub Pages in the repository settings.
6. Replace the sample data updater with an authorized live listing feed.

## Important
This package does **not** include live MLS access. To display real listings, you need an authorized data source and must follow MLS / IDX compliance rules, including attribution, source, disclaimer, and media-permission requirements.
